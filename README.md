# Portfolio React + shadcn/ui

A single-page portfolio built with React, Vite, Tailwind CSS, and shadcn/ui-inspired components. It showcases a hero introduction, experience timeline, project highlights, skill groups, and a contact call-to-action.

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to view the site.

## Available Scripts

- `npm run dev` – start the development server with hot module replacement.
- `npm run build` – type-check and produce a production build.
- `npm run preview` – preview the production build locally.
- `npm run lint` – run ESLint against the project files.
- `npm run test` – execute Vitest in run mode (no tests are defined yet, so this completes instantly).

## Tech Stack

- [React](https://react.dev) + [Vite](https://vitejs.dev)
- [Tailwind CSS](https://tailwindcss.com)
- [shadcn/ui](https://ui.shadcn.com) style primitives
- [Lucide Icons](https://lucide.dev)
